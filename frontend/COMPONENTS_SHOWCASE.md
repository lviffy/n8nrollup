# Shadcn/UI Components Showcase

A comprehensive demonstration of all available shadcn/ui components in the application.

## Accessing the Showcase

Visit `/components-showcase` or click the "View Components" button on the homepage or My Agents page.

## Components Not Demonstrated

Some components require more complex integration or are specialized:

- **Sidebar** - Complex sidebar layout component (requires full layout setup)
- **Chart** - Chart components (requires data visualization setup)
- **Sonner** - Alternative toast library (toast is already used)
- **Item** - Helper component used internally by other components

## Usage

Each component is demonstrated with:
- Basic usage examples
- Multiple variants
- Different sizes
- Interactive states
- Real-world use cases

## Navigation

The showcase is organized into 5 tabs:
1. **Buttons & Actions** - Interactive controls
2. **Forms** - Input and selection components
3. **Display** - Content presentation
4. **Overlays** - Modals and floating content
5. **Feedback** - User notifications and states

## Development

All components are located in `/components/ui/` and follow the shadcn/ui conventions:
- Fully typed with TypeScript
- Styled with Tailwind CSS
- Accessible by default
- Customizable via props and CSS variables
